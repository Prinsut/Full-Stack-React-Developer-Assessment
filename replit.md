# Full-Stack React Developer Assessment - Replit Project

## Project Overview
Complete full-stack assessment project with React, Express, MongoDB, JWT authentication, and real-time Socket.io chat.

## Important Note
This project is designed to run in **Visual Studio Code** on a local machine with MongoDB installed. It **cannot run in Replit** because MongoDB is blocked in the Replit environment.

## Project Structure
- **Backend**: Express.js server with MongoDB, JWT auth, and Socket.io
- **Frontend**: React SPA with Tailwind CSS, React Router, and Socket.io client
